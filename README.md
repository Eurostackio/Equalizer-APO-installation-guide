# Установка и настройка Equalizer APO для микрофона

> [!TIP]
># 1. Установка Equalizer APO
>
>Скачайте и установите Equalizer APO:
>
>• Официальный сайт: https://sourceforge.net/projects/equalizerapo/
>
>Во время установки:
>
>1. Поставьте галочку **Do not create shortcuts**.
>2. В окне **Configurator** найдите раздел **Capture Devices**.
>3. Выберите микрофон, который будете использовать.
>4. Завершите установку.
>
>> После установки **не перезагружайте компьютер**.

---

# 2. Первый запуск Configuration Editor

Перейдите в папку установки Equalizer APO:

```text
C:\Program Files\EqualizerAPO
```

Запустите файл:

```text
Editor.exe
```

### Предупреждение APO not installed to device

При первом запуске может появиться сообщение:

```text
APO not installed to device
```

Это нормально.

Причина в том, что во время установки был выбран только микрофон, а устройства воспроизведения (наушники или колонки) не были выбраны.

Если данное окно появляется при запуске, просто нажимайте **No**.

---

# 3. Очистка стандартной конфигурации

По умолчанию могут быть добавлены следующие элементы:

* Preamplification
* example.txt
* Graphic EQ

Чтобы удалить их:

1. Выделите нужный элемент.
2. Нажмите кнопку **Remove** (красный минус).
3. Удалите все стандартные элементы.

---

# 4. Изменение Resolution

В нижней части окна найдите раздел **Analysis Panel**.

Возле параметра **Resolution** измените значение на:

```text
1024
```

---

# 5. Выбор микрофона в Editor

1. Нажмите зелёный **+**.
2. Выберите:

```text
Control → Device (Select device)
```

3. Нажмите **Change**.
4. Снимите галочку **Select All Devices**.
5. В разделе **Capture Devices** отметьте свой микрофон.
6. Нажмите **OK**.

---

# 6. Добавление конфигураций (.txt)

Папка для конфигурационных файлов:

```text
C:\Program Files\EqualizerAPO\config
```

Поместите туда нужные `.txt` файлы.

---

# 7. Добавление VST-плагинов (.dll)

Папка для VST-плагинов:

```text
C:\Program Files\EqualizerAPO\VSTPlugins
```

Скопируйте туда необходимые `.dll` файлы.

---

# Создание собственной конфигурации

# 1. Создание TXT-файла

Откройте папку:

```text
C:\Program Files\EqualizerAPO\config
```

Создайте новый текстовый документ:

```text
ПКМ → Создать → Текстовый документ
```

Задайте любое название и сохраните файл.

Например:

```text
MyConfig.txt
```

---

# 2. Подключение конфигурации

Если Editor закрыт, запустите:

```text
C:\Program Files\EqualizerAPO\Editor.exe
```

Далее:

1. Нажмите зелёный **+**.
2. Выберите:

```text
Control → Include (Include configuration file)
```

3. Нажмите на значок папки.
4. Выберите созданный ранее `.txt` файл.
5. После добавления перетащите блок **Include** вверх списка.

> Блок должен находиться ниже **Selected Devices**, но выше остальных эффектов и плагинов.

---
> [!NOTE]
># Настройка VST-плагинов
>
># ReaGate
>
>Рекомендуемые параметры:
>
>| Параметр                | Значение  |
>| ----------------------- | --------- |
>| Pre-open                | 2 ms      |
>| Attack                  | 2 ms      |
>| Hold                    | 80 ms     |
>| Release                 | 100 ms    |
>| Hysteresis              | +2.0 dB   |
>| Lowpass                 | 8000 Hz   |
>| Highpass                | 100 Hz    |
>| RMS Size                | 1 ms      |
>| Send MIDI on open/close | Выключено |
>| Note                    | 0         |
>| Channel                 | 1         |
>
>---
>
># LoudMax
>
>Рекомендуемые параметры:
>
>| Параметр  | Значение |
>| --------- | -------- |
>| Threshold | -13.0 dB |
>| Output    | 1.0 dB   |
>| Link      | Включён  |
>| ISP       | Включён  |

---
